# vcokltfre/cascade-bot

### Cascade is a multipurpose Discord bot with weather info

Config structure in `config/config.py`:
```py
name = 'Cascade'
log_level = 'info'
log_type = 'text'
token = 'token'
hook = 'webhook_url'
dev_ids = [discord_id]

owm_token = "your_owm_token"
```
